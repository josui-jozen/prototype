# Note

## 公式ルール

- 本文
  - 50文字以内のリード文
  - 目次
  - 5行で改行
  - 1000文字未満ならセクション不要
  - 1000文字以上ならセクション分割
- 見出し画像が設定されているか
  - Figma生成
  - 内容との関連性
  - 他記事との統一感
  - サイズは2:1
- タイトル
  - 25字以内
  - 具体性
  - 検索キーワード
  - 漢字とカナが多すぎない
  - 誇大表現禁止
- Xへの投稿
  - 初めの文章を載せればいいと思う
- タグ
  - 5個つける
  - #日記, #エッセイ, #イラスト, #漫画, #小説, #音楽, #写真, #詩, #レシピ, #SF, #学び, #暮らし, #ホラー, #旅行, #恋愛, #歴史, #アニメ, #ゲーム
