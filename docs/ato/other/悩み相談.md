# 悩み相談

## 現状の整理

### やっていること

- マークダウンエディターを個人開発（約1年、エディター本体は半年）
- 先月リリース・発信済み
- モチベーションと時間は問題ない

### 悩み

- 作っても使われない、SNSが無風
- 自分でも「これなら普段使いする！」という確信が持てない
- 「Notionでよくない？」と言われて反論できない
- 差別化がブレブレ、完成の定義もわからない、迷走中

## 開発の経緯と軸のブレ

1. **最初の動機：** VSCodeで日記を書いていたが認知コストが高い → AIとGit機能のある、認知コストの低いエディターを作った
2. **転機：** 認知コストを低くしたら、たまたま設定した明朝フォントが「エモくて」気に入った
3. **迷走：** 反応がない → 差別化を探す → 組版・見た目に軸がブレる → 自分でも価値がわからなくなる

## 会話で見つかった本当の価値

### 自分が感じていること

- 明朝で書いていると、**自分の気持ちや思考に集中できる**
- **物語の中に入ったような静寂**がある
- Notionは行間が詰まってて読みにくい、太字が足りない、しっぽり明朝じゃない → **全部気に入らない**

### プロダクトの本質

> **「書く道具」ではなく「書く空間」を作っている**

- Notionは書いた**後**のためのツール（情報整理）
- Obsidianはナレッジベース（知識管理）
- このエディターは書いてる**間**のためのツール（没入・静寂・集中）

### ターゲット

- **感情で文字を書く人**
- ポエム、エッセイ、日記を書きたい人
- Obsidianが好きなビジネスマンではない

## 発信の問題

### リリース記事で訴求していたこと

> 「スマホでもマークダウンが書ける」「Google Keepからの転記が面倒」「ブラウザで使える」

→ **機能の話**。これだとNotionに負ける。

### 本当に伝えるべきだったこと

> 「書いてる間の静寂」「感情に集中できる空間」「しっぽり明朝の美しさ」

→ **体験の話**。これは機能比較の土俵に乗らない。

### 2つ目の感謝記事のほうが本質を語っていた

- 「言語化が好き」「思考や悩みを書き綴る」「感情の高ぶりを文字で表現して読み返す」
- ただし「ポエムです」と自分で格下げしてしまっている

## 結論

**プロダクトの問題ではなく、伝え方の問題。**

- 価値はある。自分自身が体感している
- でも外に出している言葉が「スマホで書けるマークダウンエディター」になっている
- 本当の価値（静寂・没入・感情への集中）をLPや発信で一度も使っていない

### 次のアクション

- 「書くことに没入できる空間」として伝えるLPを作る
- 機能ではなく体験で訴求する
- ターゲット（感情で書く人）がいる場所を見つけて届ける
