# 僕がNotionでポエムを書かなくなったわけ

僕のお仕事はプログラミングなのですが、それと同じくらい、日常的に文字を書いております。学生時代から、僕のキャンパスノートには、人生や社会についての先鋭的すぎる主張がびっしり書き込まれていました（つまり黒歴史なので詳細は伏す）。それから10年、社会はアナログからデジタルへと変容し、僕のノートはNotionに変わりました。

そんな僕の「書く」ライフワークですが、最近少し不満というか、違和感を感じるようになりました。それがタイトルにもある、「Notionでポエムを書かない」問題です。

例えばNotionを書いていると、なんだか日本語フォントが整っていないように感じます。また、メニューを開くとファイル一覧が見えますが、ミーティングやらNotionAIやらといった文字列が並んでいるのを見ると、これもなんだかポエムを書く雰囲気を損なっているように感じる。

つまり、エモくないのです。自分の中の新鮮な感情が、萎んでしまったように感じる。自分の吐き出した文字や感情に没入できない、そんな違和感を抱きました。

ところで、僕は常々、文字には2種類の役割があると思っています。情報を表す役割と、感情を表す役割です。

これまで僕が使ってきた、NotionやVSCode（最近だとObsidianもありますね）などは、どれも蓄積・整理・管理に重きを置いています。つまり、情報としての文字を扱っているのです。実際、これらのエディターは、業務効率化やナレッジ蓄積のような、ビジネス的な文脈で登場します。

一方で、感情を伝えるためのエディターというものは、まだ少ないように思います。近年は、Noteによって個人がカジュアルにポエムやエッセイを書くようになり、さらに文字を無尽蔵に生み出せるAIの台頭によって、情報そのものの価値は薄れ、その背景にある個人の思想や体験に価値がシフトしているように見えます。

情報から感情へ。そんな時代だからこそ、感情としての文字を大切にするエディターがあってもいいと思ったのです。

僕はこれまで、半年ほどの個人開発の中で、理想のエディターを追い求めてきました（詳細はnoteの過去の投稿をご覧ください）。まだ納得のいくものはできていないのですが、開発の中で発見がありました。

それは、自分好みのUIで表示された自分の文章を見ると、もっと書きたい気持ちになるということです。しっぽり明朝で書かれた自分の文章は、まるで1つの小説作品のように見えました。メニュー表示のない削ぎ落とされた画面は、静寂の中で自分の感情と向き合っているようでした。

もっと単純に、もっと自由に、ただ感じたままに書き、そうやって生まれた自分だけの表現を愛せるようなエディター。そんなものを作りたいと思っています。
