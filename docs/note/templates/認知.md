# Note公式ガイド：認知を広げるためにやるべきこと

出典：noteヘルプセンター「noteの楽しみ方」

---

## 1. 自分を知ってもらう（プロフィール周り）

### プロフィール欄を入力する
- クリエイターページは読者が最初に見る場所
- 自己紹介することで、人となりを知った上で読んでもらえる → メッセージが伝わりやすくなる
- 仕事の紹介、趣味、好きなもの、投稿頻度などを**140文字以内**で書く
- 作品を販売している場合はリンクを貼るのも有効

### プロフィールアイコンを設定する
- note上の「自分の顔」になる
- プロフィール文はクリエイターページでしか見れないが、**アイコンは記事ページにも表示される**
- どんな雰囲気の人かをパッと伝えられる

### X（Twitter）連携をする
- noteをフォローした人に、SNSアカウントもお知らせできる
- より多くの接点を作れる

### スキのお礼メッセージを設定する
- 読者がスキを押したときに任意のリアクションを表示できる
- 複数設定するとランダム表示（最大10個）
- 設定：アカウント設定 → リアクション → スキのお礼メッセージ

### フォローのお礼メッセージを設定する
- フォロー時に任意のリアクションを表示できる
- 複数設定可能（最大10個）
- 設定：アカウント設定 → リアクション → フォローのお礼メッセージ

### チップ（サポート）エリアに説明文を追加する
- 記事下のチップ案内エリアに説明文を追加できる（140文字以内）
- 説明文の横にプロフィールアイコンが表示される

### 「プロフィール」記事を登録する
- 記事をクリエイターページの「プロフィール」として固定表示できる
- 自己紹介、経歴、過去の記事リンク、自著のリンクなどに使える
- 設定：クリエイターページ → 記事の「…」→「プロフィールとして表示」

### 「仕事依頼」記事を登録する
- 記事をクリエイターページの「仕事依頼」として固定表示できる
- 設定：クリエイターページ → 記事の「…」→「仕事依頼として表示」

### 「他のクリエイターのおすすめ記事を表示する」をオンにする
- 自分の記事下に他のクリエイターの記事が表示される
- 代わりに、他のクリエイターの記事下に自分の記事が推薦される
- 設定：アカウント設定 → ユーザー設定 → チェック

## 2. 記事を読んでもらう（投稿テクニック）

### 見出し画像をアップロードする
- **「見出し画像がある記事は読まれやすい」というnote公式の統計がある**
- 見出し画像の有無でビュー数やスキ数に**数十パーセントの差**がつく
- **全ての記事に見出し画像をつけることを推奨**
- 自作が難しければ「みんなのフォトギャラリー」で他クリエイターの画像を使える

### ハッシュタグをつける（5つが効果的）
- note内で他の人に発見される機会が増える
- 人気タグ一覧（https://note.com/hashtag）を参考に
- タグ付けの4つのポイント：
  1. **記事タイプ**：#日記, #エッセイ, #ポエム, #小説 など
  2. **テーマ**：#暮らし, #恋愛, #学び など
  3. **シリーズ名・商品名**：固有名詞
  4. **関連ワード**：テーマに関連するキーワード

### SNSにシェアする
- noteフォロワーだけでなくX/Facebookにもシェアして新しい読者にリーチ
- **タイトルとURLだけでなく、記事の要約やコメントも添える**のがおすすめ
- 「この記事面白そう」と思ってクリックしてもらうのが目的

### 記事冒頭にリード文を書く（長い記事の場合）
- 要点や目次を書いたシンプルなリード文を冒頭に
- **目安は50文字程度**
- X（Twitter）でも概要が省略されずに表示される長さ

### こまめに段落を分け見出しを入れる（長い記事の場合）
- 一文が複数行にわたる場合は句読点で分割
- **5行ほどで改行や改段落**をする
- 長すぎる段落は離脱率を高める

### 「外部サイトへの配信を許諾する」をオンにする
- SmartNewsやLINE NEWSなどへの配信、出版社への紹介が行われる可能性がある
- 設定：アカウント設定 → ユーザー設定 → チェック

### コンテスト・お題企画に参加する
- noteが随時開催する企画に参加することで露出が増える
- 「#自己紹介」など書きやすいお題もある
