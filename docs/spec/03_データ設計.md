# データ設計

## ER図

```
users
  ├─< articles (author)
  │     ├─< article_versions
  │     ├─< article_tags >─ tags
  │     └─< letters
  └─< subscriptions (subscriber / author)

stamps (システム固定)
```

## テーブル定義

### users

| カラム | 型 | 制約 | 説明 |
|--------|----|------|------|
| id | uuid | PK, default gen_random_uuid() | |
| google_id | text | UNIQUE, NOT NULL | Google OAuth の sub |
| email | text | UNIQUE, NOT NULL | |
| display_name | text | NOT NULL | 表示名 |
| avatar_url | text | | Google プロフィール画像 |
| created_at | timestamptz | NOT NULL, default now() | |
| updated_at | timestamptz | NOT NULL, default now() | |

---

### articles

| カラム | 型 | 制約 | 説明 |
|--------|----|------|------|
| id | uuid | PK, default gen_random_uuid() | |
| user_id | uuid | NOT NULL, FK → users.id | 筆者 |
| title | text | NOT NULL | |
| content | text | NOT NULL | Markdown |
| visibility | text | NOT NULL, check('private','public','unlisted') | 公開範囲 |
| unlisted_token | uuid | UNIQUE | unlisted 時のアクセストークン |
| published_at | timestamptz | | 初回公開日時 |
| created_at | timestamptz | NOT NULL, default now() | |
| updated_at | timestamptz | NOT NULL, default now() | |

**visibility の意味**

| 値 | 説明 | 要件 |
|----|------|------|
| private | 自分だけ閲覧可 | PUB-01 |
| public | 全体公開 | PUB-01 |
| unlisted | unlisted_token を知る人のみ閲覧可 | PUB-02 |

---

### article_versions

バージョン上限（100件）は usecase 層で管理する。上限超過時は最古バージョンを削除する。

| カラム | 型 | 制約 | 説明 |
|--------|----|------|------|
| id | uuid | PK, default gen_random_uuid() | |
| article_id | uuid | NOT NULL, FK → articles.id | |
| content | text | NOT NULL | 保存時点の本文 |
| version_number | integer | NOT NULL | 記事内で連番（1始まり） |
| created_at | timestamptz | NOT NULL, default now() | |

UNIQUE (article_id, version_number)

---

### tags

| カラム | 型 | 制約 | 説明 |
|--------|----|------|------|
| id | uuid | PK, default gen_random_uuid() | |
| name | text | UNIQUE, NOT NULL | タグ名（表示用） |
| slug | text | UNIQUE, NOT NULL | URL用 |
| created_at | timestamptz | NOT NULL, default now() | |

---

### article_tags

| カラム | 型 | 制約 | 説明 |
|--------|----|------|------|
| article_id | uuid | NOT NULL, FK → articles.id | |
| tag_id | uuid | NOT NULL, FK → tags.id | |

PK (article_id, tag_id)

---

### subscriptions

筆者の購読（フォロー）。フォロワー数は公開しない（要件上、集計は行わない）。

| カラム | 型 | 制約 | 説明 |
|--------|----|------|------|
| subscriber_id | uuid | NOT NULL, FK → users.id | 購読者 |
| author_id | uuid | NOT NULL, FK → users.id | 購読対象の筆者 |
| created_at | timestamptz | NOT NULL, default now() | |

PK (subscriber_id, author_id)
CHECK (subscriber_id <> author_id)

---

### stamps

システム固定。アプリケーション初期データとして投入する。

| カラム | 型 | 制約 | 説明 |
|--------|----|------|------|
| id | uuid | PK, default gen_random_uuid() | |
| code | text | UNIQUE, NOT NULL | 識別コード（例: `heart`, `clap`） |
| image_url | text | NOT NULL | Cloudflare R2 上の画像URL |
| sort_order | integer | NOT NULL | 表示順 |

---

### letters

ログイン不要で送信できる感想レター。

| カラム | 型 | 制約 | 説明 |
|--------|----|------|------|
| id | uuid | PK, default gen_random_uuid() | |
| article_id | uuid | NOT NULL, FK → articles.id | 対象記事 |
| stamp_id | uuid | FK → stamps.id | 添付スタンプ（任意） |
| body | text | | 自由記入のメッセージ（任意） |
| created_at | timestamptz | NOT NULL, default now() | |

CHECK (stamp_id IS NOT NULL OR body IS NOT NULL) — スタンプとメッセージの少なくとも一方は必須

---

## インデックス方針

| テーブル | カラム | 理由 |
|----------|--------|------|
| articles | user_id | 筆者の記事一覧取得 |
| articles | unlisted_token | unlisted アクセス |
| articles | visibility, published_at | タイムライン・公開記事一覧 |
| article_versions | article_id, version_number | バージョン一覧・最古削除 |
| subscriptions | author_id | タイムライン（購読筆者の記事取得） |
| letters | article_id | 記事への感想一覧 |
