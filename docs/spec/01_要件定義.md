# 要件定義

## 概要

※これはユーザーニーズ理解のための、コピープロトタイプ実装です。

SNSのような賑やかな広場ではなく、個人的な部屋のような空間を提供する。日記やエッセイなど、有益性より個人的な記録・表現を重視する。

- **必要最小限の機能** — コミュニティ機能を意図的に排除し、書くことに集中できる環境を提供する
- **静かなつながり** — フォロワー数やいいね数を可視化せず、控えめな手段でのみ他者とつながる

## 機能要件

### 認証

| ID | 機能 | 説明 |
|----|------|------|
| AUTH-01 | Googleログイン | Googleアカウントによるソーシャルログイン |

### エディタ

| ID | 機能 | 説明 |
|----|------|------|
| EDIT-01 | リッチテキスト編集 | マークダウンベースのリッチテキストエディタ |
| EDIT-02 | 画像挿入 | ボタンまたはドラッグ&ドロップによる画像挿入 |
| EDIT-03 | コンテンツ埋め込み | YouTube、X、Bluesky、リンクカードの埋め込み |
| EDIT-04 | タグ付け | 記事へのタグ付け |
| EDIT-05 | バージョン管理 | 本文の編集履歴を保持（100件まで遡行可能） |
| EDIT-06 | ムード変更 | エディタの雰囲気（テーマ）を変更（PC限定） |
| EDIT-07 | スタンプ | 記事内で使用できるスタンプ機能 |

### 公開・閲覧

| ID | 機能 | 説明 |
|----|------|------|
| PUB-01 | 公開範囲設定 | 「自分だけ」「みんな」の2種類 |
| PUB-02 | 限定公開 | URLを知っている人のみ閲覧可能 |

### 購読・フィード

| ID | 機能 | 説明 |
|----|------|------|
| FEED-01 | タイムライン | 購読した筆者の更新一覧を表示 |

### リアクション

| ID | 機能 | 説明 |
|----|------|------|
| REACT-01 | 感想レター | ログイン不要で定型メッセージを筆者に送信 |

### API

| ID | 機能 | 説明 |
|----|------|------|
| API-01 | 記事操作API | 外部からのAPI経由での記事の取得・投稿・更新・削除 |

### 実装しない機能(コンセプト違反のため)

- いいねボタン
- コメント欄
- フォロワー数の表示
- リポスト / シェア数の表示
