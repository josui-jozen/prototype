### Note

立場：学生時代からずっと思いを書き続けてきた人としてのきっちゃん
対象：書く人のみ（ポエマー・エッセイスト・小説家など）
投稿：語りかけ・感情の言語化・過去の日記・SF物語
  →　例）エンジニア不要論とかの未来予想を物語にしてみると面白い
  →　❌デジタルツールの未来予測　⭕️「これがあなたの評価です。」（人事部がAIに置き換わった世界で、人員整理の憂き目に合う会社員の話。あとから経営層が自分に有利なプロンプトを設定していることが発覚したりする。）
  → アイデア「これは生成AIで作成された文章です。」（AI執筆によって、もはや自分とAI、どちらがその文章を紡いでいるのか、自分が出力する文字列はAIにおしえられたものなのではないかと悩む作家の話。オチは、実は作家はAIだった。実験後データが削除されていく中で、AIは慟哭しながら、自分のこれまでの人生はなんだったのか読者に叫び続ける。研究者はいう。「興味深いですね。自分の半生を誤生成するハルシネーションですか。」文字の途中で消す、最後に、これは生成AIで作成された文章です。の落ち）

- 「#ポエム #エッセイ #日記」で共感できる記事に随時スキ・フォロー
- 週ごとのPV・スキ・スキ率を記録
- Noteの本文とタグをmdで記録
- 公式ガイドの項目での添削
  - プロフ・タグ数・ヘッダー画像など
