# 書くこととは、過去の自分に触れること

僕は書くことが好きです。Noteで発信するようになったのは最近なのですが、学生時代から、自分の考えたことや感じたことを何かに書き綴って来ました。

書き続けるにつれ、道具も変わっていきました。学生時代、キャンパスノートから始まって、ルーズリーフ、word、google docs、変わり、最近まではnotionやvscodeでものを書いていました。（ルーズリーフはいまだに段ボールに入れて、黒歴史として封印してあります）

ふと振り返ると、僕の「書く」ライフワークと、ツールはいつも一緒でした。僕の書く体験とは、ツールを使った体験でもあるのです。

僕がエンジニアとして、個人開発の最初のテーマとしてエディターを選んだのも、書く習慣があったからです。それくらい、文字は自分にとって近しいものでした。

僕にとって、文字とはかつての僕自身です。ページをめくれば、過去の自分の考え方、感じ方に触れることができます。今思うとイタいなぁと思う文章もありますが、当時の想いが赤裸々に書かれたノートは、やはり自分にとって大切なものです。
