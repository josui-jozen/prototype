# LP構成案

## 前提

- 信念系プロダクトのため、機能説明ではなく「共鳴」と「体験」で訴求する
- Note分析より、信念・感情系コンテンツのスキ率11.9%に対し機能紹介は0%。機能を語っても刺さらない
- X経由の流入はほぼゼロ（URLクリック0）。現状の主要流入元はNote検索・タイムライン
- ターゲットはNote上の「書く層」（ポエマー・エッセイスト・日記書き）。開発者層ではない

## 構成

### 1. キャッチコピー + ビジュアル

- 一文で世界観を伝える（例：「気持ちを書く自分を、好きになる。」）
- しっぽり明朝で組まれた文章のスクリーンショットなど、プロダクトの空気感が伝わるビジュアル

### 2. 問題提起

- 「Notionでポエムを書く気にならない」共感から入る
- Note記事（ビュー73, スキ率11%）で検証済みの切り口をそのまま使う

### 3. 信念の表明（マニフェスト）

- 文字には「情報伝達」と「自己表現」の2つの役割がある
- 既存ツールは情報側に最適化されている。感情側のエディターがない
- 「あと、」は書き手の感情表現とその肯定を大切にする

### 4. 体験版（登録不要）

- LPに埋め込み、その場で書ける
- 体験ループを1回まわしてもらう：書く → 自分好みに表示される → 愛着が湧く
- UIカスタム（フォント・行間・背景）と日本語組版を体感させる

### 5. CTA

- ウェイトリスト登録 or アプリへの導線

## 本文

### ファーストビュー

僕を書くためのエディター。自分だけの表現を、自分だけの見た目で書く。自分のありのままの感情を愛するための執筆道具。

### 思想

「エディターを作りたい。」

AI全盛のこの時代、多くの人々が文字を書くようになった時代。ナレッジ蓄積、AI活用。エディターは、より優れた情報処理を目指している。より優れたワークスペースを目指している。

ちょっと疲れていませんか？

だからこそ、思いました。情報としての文字ではなく、感情としての文字を大切にしたいと。

### 機能

- UIカスタム
    - フォント・行間・サイズ・背景。書く環境を細かく調整できます。
- 日本語組版
    - 海外エディターには真似できない、日本語に最適化されたUI。
- 集中できるUI
    - 無駄な機能はなくし、あなたの文章を美しく表示することに最適化します。
- いつでもどこでも
    - スマホでもパソコンでも、オンラインでもオフラインでも。あなたの感情を、その瞬間にかける、あなただけの場所になります。
