# 体験版エディターのデザイン指示

LP内の思想セクションと価値セクションの間に埋め込む、体験版エディターのデザインをお願いします。

## 目的

ユーザーに「書く → 自分好みの見た目で表示される → もっと書きたくなる」の体験ループを1回だけまわしてもらうこと。登録不要で、その場で書ける。

## 画面構成

### デフォルト状態
- エディター領域のみが表示されている
- プレースホルダー：「ここに、なにか書いてみてください。」
- 右上にメニューボタン（縦三点 ⋮）が控えめに配置
- それ以外のUIは一切なし。書くことだけに集中させる

### メニュー展開時
- メニューボタンを押すと、右からスライドでメニューパネルが現れる
- メニュー内容：
  - **フォント切り替え**（3択）
    - しっぽり明朝（デフォルト）
    - 游明朝
    - ゴシック
  - **背景色切り替え**（3択）
    - 白練 #fcfaf2（デフォルト）
    - 墨 #373737
    - 薄藍 #e8f0f8
- フォント・背景を切り替えると、エディター内の文字がリアルタイムで変化する
- 墨背景の場合、文字色は白練に自動反転する

## デザイン方針

- LPの一部として自然に溶け込むこと。独立したアプリのように見えてはいけない
- エディター領域とLPの境界線は極力なくす
- メニューパネルはエディター領域内に収まる（外にはみ出さない）
- メニューパネルの幅はエディター領域の1/4程度
- メニュー内のオプション（フォント名、背景色スウォッチなど）は幅に収まらない場合は自動で折り返す（flex-wrap）
- メニューは半透明＋バックドロップブラー
- テーマカラー：白練 #fcfaf2 / 墨 #373737 / 天 #0083c6
- フォント：しっぽり明朝（本文）
- 装飾は最小限。ボタンもテキストベースで、派手な色やアイコンは使わない

## サイズ感

- 幅：LP本文と同じ幅（max-width: 560px）
- 高さ：最低300px程度。書く量に応じて伸びる
- スマホファーストで設計

## 注意点

- これはLPの体験版であり、あと、本体のエディターとは別物
- 全機能を見せる必要はない。「書いたら綺麗に表示される」体験だけ伝われば十分
- 複雑に見えた瞬間、ユーザーは書かない
